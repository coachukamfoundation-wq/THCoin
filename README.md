# THCoin

THCoin Mini App

THCoin is a Telegram Mini App where users can earn THCoins by completing daily missions, watching rewarded ads, and participating in interactive activities. The app features a modern user interface, user profiles, a rewards system, and future support for blockchain integration and USDT rewards

.

Features

- 🎯 Daily missions
- ⭐ THCoin rewards
- 📺 Rewarded ads
- 👤 User profile system
- 💰 Earn USDT section
- ⚡ Fast and mobile-friendly design
- 🔒 Secure backend integration (coming soon)



This project is under active development, and new features will be added regularly.
